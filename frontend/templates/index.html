About TL-IDS

Final Year B.Tech Project · Department of Computer Science & Engineering · 2024–25

Project Overview

Wireless Sensor Networks (WSN) are highly vulnerable to cyber attacks due to constrained resources, open wireless channels, and absence of dedicated security hardware.


This project proposes a Transfer Learning-based Intrusion Detection System (TL-IDS) that overcomes the labeled data shortage in WSN by transferring knowledge from four well-studied network intrusion datasets.


Using domain-adversarial adaptation and gradual fine-tuning, the model detects 6 WSN attack categories with 98.7% accuracy, 0.985 F1-score, and only 0.3% false positives — running at 12ms inference latency on embedded gateway hardware.
